# Alex Ashing Portfolio

A modern, responsive portfolio website showcasing work across music, visuals, and technology.

## Project Structure

```
webv2/
├── index.html          # Homepage with hero and discipline tiles
├── about.html          # About page with philosophy and timeline
├── music.html          # Music portfolio page
├── visuals.html        # Visual work portfolio page
├── tech.html           # Technology/development portfolio page
├── styles.css          # Unified stylesheet
└── README.md           # This file
```

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Unified Navigation**: Consistent navigation across all pages
- **Modern CSS**: CSS custom properties for consistent theming
- **Interactive Elements**: Hover effects and smooth transitions
- **Professional Typography**: Playfair Display + DM Sans font combination

## Color Scheme

- **Primary**: Gold (#C9A84C) - branding and accents
- **Music**: Gold theme
- **Visuals**: Teal theme (#6BA8A4)
- **Tech**: Red theme (#C84B4B)
- **About**: Blue theme (#7A8CFF)
- **Background**: Dark (#080705) for professional look

## Navigation Structure

- **Home**: Landing page with overview
- **Music**: Music performance and production
- **Visuals**: Photography and visual work
- **Tech**: Development and technical projects
- **About**: Background and philosophy
